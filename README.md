# 🐍 Scraper to compare prices for hair care products

## 📄 Sites scraped:

- [Emag.bg](https://www.emag.bg/search/remington/p1)
- [Notino.bg](https://www.notino.bg/remington/?f=1-1-17015)
- [Technomarket.bg](https://www.technomarket.bg/search?query=remington)
- [Technopolis.bg](https://www.technopolis.bg/en/search/?pricerange=&pageselect=90&page=0&q=remington:relevance&text=remington&layout=List)
- [Zora.bg](https://zora.bg/search?per_page=50&query=remington)

### 🔥 Merger.py

This is the script that merges all the data scraped from these sites into one `.csv` file with proper indexing and weeding out the duplicate products.
